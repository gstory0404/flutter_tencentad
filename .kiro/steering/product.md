# Product Overview

flutter_tencentad is a Flutter plugin that integrates Tencent's GDT (广点通/优量汇) advertising SDK for Android, iOS, and OHOS (HarmonyOS) platforms.

## Purpose
Provides a unified Dart API for displaying various ad formats from Tencent's advertising network in Flutter applications.

## Supported Ad Types
- Splash ads (开屏广告) - Full-screen ads shown at app launch
- Banner ads (横幅广告) - Standard banner advertisements
- Native express ads (信息流广告) - Dynamic feed/video ads
- Reward video ads (激励视频广告) - Video ads with user rewards
- Interstitial ads (插屏广告) - Full or partial screen overlay ads

## Key Features
- Bidding support (竞价模式) for all ad types
- Privacy compliance controls for Android
- Personalized ad settings
- Channel ID configuration
- Download confirmation dialogs
- Event callbacks for ad lifecycle management

## Target Users
Flutter developers integrating Tencent advertising into Chinese market applications.

## Related Projects
Part of a family of ad plugins by gstory, including GTAds for unified multi-platform ad management.
